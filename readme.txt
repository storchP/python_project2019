【M@ster Widget for windows powerd by open widget project】

一定のシナリオに沿って、双方にとって有用なリンクを順番別に行う事を想定しています。

WindowsXP〜womdpws10までのOSかつ、32bit,64bit双方で動作しますが、利用者の環境上による問題が起きる可能性もございます。
